# car_num_check
车牌识别系统，基于Python实现，通过TensorFlow搭建CNN卷积神经网络模型，对车牌数据集图片进行训练最后得到模型，并基于Django框架搭建网页端平台，实现用户在网页端输入一张图片识别其结果，并基于Pyqt5搭建桌面端可视化界面。
